# PeachlifeAB/brew

Public Homebrew tap for PeachlifeAB tools.

## Install

```bash
brew tap PeachlifeAB/brew
```

Then install any tool:

```bash
brew install --cask hyprspace
brew install sive
brew install lgtvctrl
```

## Tools

| Tool | Type | Description |
|---|---|---|
| hyprspace | cask | Tiling window manager based on AeroSpace |
| sive | formula | Sync secrets from your vault into your shell |
| lgtvctrl | formula | Command-line control for LG WebOS TVs |

## Update

```bash
brew upgrade --cask hyprspace
brew upgrade sive
brew upgrade lgtvctrl
```

## Uninstall

```bash
brew uninstall --cask hyprspace
brew uninstall sive
brew uninstall lgtvctrl
brew untap PeachlifeAB/brew
```
