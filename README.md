# brew

Public Homebrew tap for Hyprspace.

## Install

```bash
brew tap PeachlifeAB/tap
brew install --cask hyprspace
```

## Update

```bash
brew upgrade --cask hyprspace
```

## Uninstall

```bash
brew uninstall --cask hyprspace
brew untap PeachlifeAB/tap
```
